DNS adalah server yang bisa melayani permintaan untuk mengetahui sebuah IP address yang digunakan oleh suatu domain. Permisalan mudahnya , saat anda ingin mengakses facebook.com maka server DNS yang akan mencari IP address dari facebook sehingga komputer anda dapat mengakses facebook. Penggunaan DNS server ini biasanya sudah disetting otomatis di setiap komputer dan setiap provider penyedia layanan internet. Namun jika belum maka perlu pengaturan di bagian IP address terlebih dahulu. Apabila anda menggunakan router, maka anda bisa membuat server DNS sendiri. Akan tetapi pencarian IP address tetap dilakukan oleh provider penyedia layanan internet.
